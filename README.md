# 🌍 World Population Analysis (2024)

## 📌 Overview

This project analyzes global population data for 2024 using the World Bank dataset.
The goal is to identify the most populated countries and understand how population is distributed across the world.

---

## 📊 Features

* Data cleaning and preprocessing using Pandas
* Extraction of top 10 most populated countries
* Calculation of world population percentage
* Visualization using bar chart and histogram

---

## 🛠️ Tools & Technologies

* Python 🐍
* Pandas 📊
* Matplotlib 📈
* Google Colab 💻

---

## 📁 Dataset

**Source:** World Bank Population Dataset (CSV)
**File:** `API_SP.POP.TOTL_DS2_en_csv_v2_61.csv`

---

## 📈 Output

* Top 10 most populated countries (2024)
* Bar chart visualization
* Histogram showing population distribution

---

## 🔍 Insights

* A few countries contribute to a large portion of the global population
* India and China dominate global population rankings
* Population distribution is highly uneven across countries
